# 参数定义

## 参数列表

| 参数名     | 类型         | 描述                 |
|------------|--------------|----------------------|
| name       | string       | 用户名               |
| age        | number       | 年龄                 |
| address    | string       | 住址                 |
| contact    | details      | 联系方式，展开查看   |
